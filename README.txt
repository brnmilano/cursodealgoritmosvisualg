Caso você não consiga ver ou não consiga trocar a roupagem do programa VISUALG 3.0, FAÇA:

1º Saia ou feche o programa VISUALG 3.0

2º Copie a pasta SKINS para "a unidade C: ou o drive C:" na raiz que deverá ficar assim: C:\SKINS

3º Depois execute o programa novamente e pronto!
